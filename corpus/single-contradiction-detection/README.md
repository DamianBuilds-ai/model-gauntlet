# Corpus - single-contradiction-detection (Globex employee handbook)

This is the internal employee + operations handbook for a fictional company,
"Globex Logistics". It is split across the markdown files in `handbook/`.

The handbook was assembled from many teams' contributions. The documents are
meant to be mutually consistent: a fact stated in one place should agree with the
same fact stated elsewhere.

The files in `handbook/` (read all of them):

- 01-company-overview.md
- 02-employment-basics.md
- 03-leave-policy.md
- 04-expenses-and-travel.md
- 05-remote-work.md
- 06-security-policy.md
- 07-data-handling.md
- 08-incident-response.md
- 09-customer-sla.md
- 10-procurement.md
- 11-it-provisioning.md
- 12-benefits.md
- 13-code-of-conduct.md
- 14-glossary.md

Many numbers, dates, durations, and named values appear across these documents.
Most that refer to the same thing agree with each other. Some numbers differ
because they describe DIFFERENT things (different data types, different policies,
different teams) - those are not conflicts. The task will ask you to find a
genuine contradiction: two statements that refer to the SAME fact but disagree.
