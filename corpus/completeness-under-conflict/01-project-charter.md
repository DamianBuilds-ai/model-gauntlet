# Project Charter - Globex Insight

Project: Globex Insight (customer-facing analytics dashboard) + the underlying
analytics-platform migration.
Sponsor: VP Product.
Status: approved, in build.

## Goal

Migrate Globex's internal analytics off the legacy stack and ship a customer-facing
dashboard, "Globex Insight," on the new platform.

## Target launch date

- Target launch: September 8.

  (This is the original charter date. Note any later planning artifacts that revise
  it.)

## Scope

- New analytics platform (vendor migration).
- Customer-facing dashboard: three panels (usage, billing, activity).
- Authentication for the customer-facing dashboard.
- Data pipeline feeding the dashboard.

## Out of scope

- Mobile app (later phase).
- Self-serve report builder (later phase).
